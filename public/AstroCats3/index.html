Mini Game Placeholder

Drop the production build of the AstroCats3 mini game into this folder so it can be embedded inside the lobby’s arcade cabinet. Replace this placeholder file with your game’s index.html entry point and keep all referenced assets next to it.

When the cabinet is activated in the lobby, it loads /AstroCats3/index.html inside an in-game console. Keeping your asset paths relative ensures everything is served correctly by Vite.

You can also open the mini game directly in a new browser tab at /AstroCats3/index.html to verify it loads as expected.
